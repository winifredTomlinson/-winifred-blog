字符类

字符类，即将直接量字符单独放进方括号内，而因为有些字符类非常常用，所以用了一些特殊字符的转义字符来表示他们

如：
\w:匹配所有ASCLL码的字符，包括数字和字母
\W：匹配所有不是ASCLL码的字符
\s：匹配空格符
\S：匹配不是空格符
\d：匹配数字
\D：匹配不是数字

JavaScript

函数的四种调用方式： 1、普通函数调用 2、作为对象的方法调用 3、作为构造函数调用 4、使用apply和call函数调用 PS:当使用第四种方式时，要注意其共性与区别，共性为都有两个参数，第一个参数为希望绑定的this对象，主要区别为apply参数要作为数组传人，加引号并用逗号隔开["",""]，而call的参数是与所调用函数的参数一一对应的。而bind和apply，call一样，都可以绑定对象，但是bind返回的仍然是一个函数，需要用()再进行调用，而后者直接是对函数的调用，故不需要()。 所谓闭包是在外部函数内部定义内部函数，内部函数调用外部函数的参数，并且在外部函数外将调用内部函数。在理解闭包时要注意理解，执行环境，作用域链以及变量常驻内存，不会被垃圾回收机制回收等情况。

函数的三种定义方式 1、通常的函数定义即通过function语句实现（也叫声明式函数） function funcname([arg1[,arg2......]]){statements} PS:函数定义描述的是一个静态的程序结构，当 JavaScript 解析器遇到一个函数定义时，它就被解析并存储构成函数体的语句，然后定义一个和该函数名同名的全局或者某个对象属性来保存。当该函数被调用时，里面的函数体才真正的被执行。如：

alert(typeof f);function f(x,y){alert(x+y);} f(2,3);

2、Function()构造函数（匿名函数） var myfunc=new Function('x','y','alert(x+y)'); 3、函数直接量 函数直接量是一个表达式，它即可以定义一个匿名函数，也可以定义一个带函数名的函数: var sum = function(a, b){ return a+b; } var f = function fact(x){ if(x<= 1){ return 1; }else{ return x * fact(x-1); } } 三种定义方式的区别 可以从作用域、效率以及加载顺序来区分。 1、从作用域上来说，函数声明式和函数直接量使用的是局部变量，而 Function()构造函数却是全局变量 2、执行效率上来说，Function()构造函数的效率要低于其他两种方式，尤其是在循环体中，因为构造函数每执行一次都要重新编译，并且生成新的函数对象，为了加快网页加载速度，提高用户体验，不建议选择 Function ()构造函数方式来定义函数。 3、加载顺序，function 方式(即函数声明式)是在 JavaScript 编译的时候就加载到作用域中,而其他两种方式则是在代码执行的时候加载，如果在定义之前调用它，则会返回 undefined：
